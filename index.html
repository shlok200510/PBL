English Text Extraction using OCR and Translation to Hindi using Transformer Model

"This project proposes an automated system that extracts English text from images using Optical Character Recognition and translates it into Hindi using a Transformer-based Neural Machine Translation model. "

Problem Statement

Many important documents in India are available only in English. A large population prefers Hindi for understanding government forms, prescriptions, and educational material. Manual translation of scanned documents is slow, expensive and error-prone. Therefore an automatic image-to-Hindi translation system is required.

Literature Review / Market Research

Early OCR systems used pattern matching, while modern OCR uses deep learning. Neural Machine Translation replaced rule-based and statistical methods. Transformer models using self-attention now achieve state-of-the-art results in translation.

Research Gap / Innovation

Existing solutions rely on internet APIs such as Google Translate. Our system is an offline research-based model that integrates OCR with a custom-trained Transformer to perform end-to-end image-to-Hindi translation.

System Methodology

Dataset / Input

English–Hindi parallel corpus (~100k+ sentence pairs). Text tokenized using SpaCy for English and IndicNLP tokenizer for Hindi. Vocabulary built with 10,000 most frequent words.

Model / Architecture

Image preprocessing → OCR text extraction → tokenization → vocabulary encoding → Transformer encoder-decoder with attention → beam search decoding → Hindi output.

Live Execution

VIEW CODE / DEMO

The model was trained for 21 epochs. Both training and validation loss decreased steadily, indicating successful learning without overfitting. The model generates grammatically correct Hindi translations and generalizes to unseen sentences.

Accuracy / Performance BLEU Score ≈ 0.48

Quantifiable outcomes and evaluation metrics compared to baselines.
